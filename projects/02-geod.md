# GEOD / GEODNET

## Overview
**Identity**
- **GEODNET** is a DePIN network for centimeter-level RTK/GNSS positioning, targeting robotics, drones, agriculture, construction, autonomous systems, and “physical AI.”
- Token: `$GEOD`.

**Official links**
- X: https://x.com/GEODNET
- Website: https://geodnet.com/
- Token page: https://geodnet.com/geod
- Docs: https://docs.geodnet.com/
- Console/map: https://console.geodnet.com/ / https://console.geodnet.com/map
- Discord: https://discord.com/invite/geodnet

**Team/key people**
- Mike Horton — founder/co-founder/CEO: https://x.com/mikeahorton
- David — co-founder, surfaced as https://x.com/mvrender

- Notable investor/KOL: Kyle Samani / Multicoin.
