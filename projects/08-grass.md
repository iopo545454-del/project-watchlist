# GRASS

## Overview
- Decentralized web-data / bandwidth network for AI data collection.
- Users run nodes/extensions to contribute unused bandwidth; network routes public web data collection toward AI/data customers.
- Token: `GRASS` on Solana.
- Builder: Wynd Labs / Wynd Network; ecosystem/governance via Grass Foundation.
