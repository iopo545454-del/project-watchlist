# VIRTUAL / Virtuals Protocol

## Overview
**Identity**
- Base-centered onchain AI-agent / agent-commerce protocol; $VIRTUAL is the ecosystem token for agent launches, co-ownership and protocol activity.

**Official links**
- X: https://x.com/virtuals_io
- Website: https://www.virtuals.io/
- App/OS: https://app.virtuals.io/ and/or https://os.virtuals.io/ (verify current canonical entry from official X)
- Docs/agent commerce: https://agdp.io/
- GitHub: https://github.com/Virtual-Protocol
- Token: Base $VIRTUAL reported as `0x4b5d32a07b8d3ec5d6928caa30196f8dd6a7c5a9` — verify from official docs/X before use.

**Team/key people**
- EtherMage / likely Jansen — co-founder/architect, https://x.com/ethermage
- Core contributors to monitor: https://x.com/umeirzz, https://x.com/0xTP91, https://x.com/everythingempty
- Historical origin: PathDAO pivot/evolution; Jansen and Wee Kee cited around PathDAO history.

**Investors/backers**
- PathDAO era: DeFiance Capital and Merit Circle led/participated in early seed/token-sale context.
- Virtuals increasingly uses ecosystem funding/ACF/launch mechanisms rather than emphasizing a traditional cap table.

**KOL/community accounts**
- `@virtuals_io`, `@ethermage`, `@umeirzz`, `@0xTP91`.
- Ecosystem-builder accounts mentioned by official posts should be added case-by-case; high impersonation risk around agent tickers.

**Data/resources to monitor**
- Base token contract, holder concentration, DEX liquidity, launch/bonding-curve flows.
- Virtuals app/Genesis/Titan/Unicorn launch pages; agent revenue and ACF metrics where published.
- GitHub `Virtual-Protocol` repos; AGDP/ACP/ERC-8183 docs.
- DefiLlama search for Virtuals / Base ecosystem liquidity; Dune dashboards if official/community-vetted.

**Alert triggers**
- Changes to launch mechanics, bonding curves, points/rewards, protocol fees.
- Large agent launches, Binance/CEX listings for ecosystem agents, security warnings.
- Standards adoption (agent commerce, wallets/payments/memory/inference) by external partners.
- Contract migrations, cross-chain liquidity moves, ownership/audit updates.

**Notes/ambiguities**
- Public team is partly pseudonymous/low-profile; verify claimed contributors through official reposts/bios.
- Token contract address should be rechecked before monitoring/transactions due frequent scam clones.
