# AEON / Aeon

## Overview
**Identity**
- Most likely current target: **AEON / AEON.XYZ**, an AI payments / agentic-economy settlement infrastructure project.
- Distinct from older **AeonCoin `$AEON`**, a legacy Monero-style privacy coin with https://x.com/AeonCoin and aeon.cash.
- For current payments context, track **AEON Community / aeon.xyz**.

**Official links**
- X: https://x.com/AEON_Community
- Website: https://aeon.xyz
- GitHub/docs repo found: https://github.com/AEON-Project/bnb-x402/
- AEON Pay bot mentioned: https://t.me/AEON_Pay_bot

**Team/key people**
- Leo — CTO/key builder, surfaced as https://x.com/alohaleonardox.
- Broader team is not highly public; official account is the main comms hub.

**Investors/backers**
- Reportedly raised **$8M led by YZi Labs**.
- Other participants cited: IDG Capital, HashKey Capital, The BB Fund, Oak Grove VC, SevenX Ventures, Alchemy, Draper Dragon, Contribution Capital, UpHonest Capital.
- Ecosystem/support: BNB Chain / MVB context.

**KOL/community accounts**
- https://x.com/AEON_Community
- https://x.com/yzilabs
- https://x.com/BNBCHAIN
- https://x.com/alohaleonardox

**Data/resources to monitor**
- GitHub repo activity for bnb-x402 SDK.
- AEON website/news for AI Gateway, x402 Facilitator, merchant integrations.
- BNB Chain ecosystem announcements.
- Merchant/payment partner announcements.
- Any official token announcement — currently no clear native `$AEON` token confirmed for aeon.xyz.

**Alert triggers**
- Native token/TGE announcement or contract publication.
- x402 integration growth, API/tool-provider partnerships.
- Merchant coverage expansion or new supported tokens.
- New funding/backer announcements.
- Security/audit reports for payment facilitator contracts.

**Notes/ambiguities**
- Major ambiguity: “AEON” can mean old AeonCoin, AEON.XYZ payments infra, or unrelated tokens. Treat any tradable `$AEON` token as unverified unless linked by @AEON_Community.
